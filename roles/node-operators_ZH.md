# 节点操作员

## 概况

闪链的网络是由闪链节点（THORNode) 组成的，起始计划是99个节点。 每一个节点在独立的服务器上运行，所有的节点进行密切的交流与合作来实现一个跨链网络。

### 匿名制度

为了保证网络和个人的安全性，节点操作员需要保持匿名性，不通过任何方式透露自己的身份。因此，闪链不支持通过代币投票或者将代币提供给第三方节点。
